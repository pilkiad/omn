# ROS 2 Jazzy LLM Wiki

This directory is a self-contained Markdown wiki for the official ROS 2 Jazzy documentation, reorganized for coding agents such as Codex.
It was generated from the `jazzy` branch of `ros2/ros2_documentation` at commit `4b4d15e49c5eb047810c641155debe90d705056d`.

Start here:

- [AGENTS.md](AGENTS.md) - operating instructions for coding agents.
- [llms.txt](llms.txt) - compact LLM entrypoint.
- [index.md](index.md) - complete content catalog.
- [SUMMARY.md](SUMMARY.md) - grouped navigation.
- [wiki/task-map.md](wiki/task-map.md) - task-oriented routes through the docs.
- [wiki/concept-map.md](wiki/concept-map.md) - core ROS 2 concepts.
- [wiki/tutorial-paths.md](wiki/tutorial-paths.md) - tutorial learning paths.
- [wiki/tooling-map.md](wiki/tooling-map.md) - project, release, package, and tooling references.
- [wiki/troubleshooting-map.md](wiki/troubleshooting-map.md) - debugging and troubleshooting routes.

Content pages live under `content/`. Local images and downloads live under `assets/`.
The upstream source checkout and Sphinx build output are not required to use this wiki.
